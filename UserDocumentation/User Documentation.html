Vaisala DigiCORA® Sounding System MW41 User Documentation

Download the relevant documents using the links below.

In addition, MW41 sounding software includes an embedded on-line help.

Note that you need the Adobe Reader program (version 5.0 or later) for viewing the user documentation. You can get the latest version easily from the Adobe website.

Adobe Reader is also available on MW41 installation media for your convenience, in case you do not have Internet connection available. The .exe file is available in folder 3rdPartySoftware.
