Scripting Interfaces

ISounding

Sounding system provides this interface to the script. In the script you can use SoundingInterface variable to access the interface methods. E.g. SoundingInterface.GetSoundings() call returns information of all soundings in the sounding system database.

IExecutableScript

Script should implement all the methods described by this interface. With SoundingInterface.OrderNotifications(...) method script can describe what data events will be notified in the HandleData calls.
