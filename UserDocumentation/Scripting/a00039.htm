Detailed Description

A generic collection for types that implement the generic IEpochData interface. The collection is a queue with some additional functionality for time-related operations.

Template Parameters
TThe type of the elements in the collection
UThe type parameter of the IEpochData interface that is implemented by all of the elements
Type Constraints
T :IEpochData<U>
