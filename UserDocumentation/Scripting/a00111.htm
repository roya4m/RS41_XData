Detailed Description

RangeFormat class for implementing the script.

Note that file should be named same as executed class.

Class should implement at least method defined by IExecutableScript interface.

RNGFMT report script

Command line options:
  -f <pathname> 
      directory for report files.
  -n
      Report files made from archived sounding. 
      Can be used for generating report manually with ScriptExecutor.exe.
