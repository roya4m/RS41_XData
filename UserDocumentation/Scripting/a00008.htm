Scripts interface version changes

Currently used IronPython version is 2.6.1 with .NET 2.0 SP1. IronPython is used under Microsoft Public License (Ms-PL).

Supported script interface version number is incremented when there is possibility that old scripts are not combatible with interface changes. Script interface changes:

  • Script interface version 1:
    • MW41 version 1.1:
      • Initial script interface.
  • Script interface version 2:
    • MW41 version 2.4:
      • In script interface references to VirtualTemperature class has been replaced with MetCalculations class.
      • Added more support for handling generated messages.
    • MW41 version 2.6:
      • Script interface supports also certain VSP events from AUTOSONDE.
      • Added option for "Run at startup" for the scripts.
  • Script interface version 3:
    • MW41 version 2.8:
      • Added SupportedScriptInterfaceVersion to IExecutableScript. All the scripts should implement this new method.
      • Recalculation feature in MW41. If your script is handling realtime data during the sounding it would be good to handle RecalculationStarted and RecalculationDone events in the script. After RecalculationStarted event all recalculated sounding data will be send again to script until RecalculationDone event is received.
    • MW41 version 2.9:
      • Added RS41 version information and in-build check values to RadioSondeData
    • MW41 version 2.11.0
      • Following metadata key information has been moved to SoundingInformation: LAUNCH_SITE_GEOPOTENTIAL_HEIGHT, LAUNCH_SITE_MSL_HEIGHT, LAUNCH_SITE_LATITUDE, LAUNCH_SITE_LONGITUDE, LAUNCH_SITE_ALTITUDE_OFFSET, LAUNCH_SITE_DISTANCE_OFFSET, LAUNCH_SITE_DIRECTION_OFFSET, ASCENT_RATE_SURFACE_TO_400HPA, ASCENT_RATE_400HPA_TO_TERMINATION, TERMINATION_HEIGHT and TERMINATION_PRESSURE.
    • MW41 version 2.13.0
      • Method calculating mixing ration has been moved to MetAlgorithms assembly. Required changes in the using CalculateMixingRatio method:
        1 Old code:
        2  clr.AddReference('MetCalculations')
        3  from Vaisala.Soundings.Framework.Calculation.PTU import MetCalculations
        4  mixratio = MetCalculations.CalculateMixingRatio(pressure, temperature, humidity)
        5 New code:
        6  clr.AddReference('MetAlgorithms')
        7  from Vaisala.Soundings.Framework.Calculation import PtuAlgorithms
        8  mixratio = PtuAlgorithms.CalculateMixingRatio(pressure, temperature, humidity)
