Detailed Description

OzoneMain class for implementing the script.

Class should implement at least method defined by IExecutableScript interface.

Ozone main script

In this script we receive notifications from sounding system and 
calculate ozone partial pressure when sounding is ongoing.

Command line options:
  -d <debug output level>
  -f <file pathname> 
     for ozone result textfile.
