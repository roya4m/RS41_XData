Detailed Description

AERO

Command line options:
  -d <message destination> 
      Add new destination where report is distributed.
      Note that destination should be created in sounding system configuration.
  -h <header> 
      Add message header to AERO report.
  -All
      Create AERO from all soundings in the database. 
      Can be used for generating report manually with ScriptExecutor.exe.
