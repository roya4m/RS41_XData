Detailed Description

WOUDC class for implementing the script.

Note that file should be named same as executed class.

Class should implement at least method defined by IExecutableScript interface.

WOUDC script

Create WOUDC report from ozone sounding. 
The script is used with ScriptExecutor.exe.
